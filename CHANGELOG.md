## 部署指引

### 1. 开通云开发环境
```bash
# 在微信开发者工具中右键 cloudfunctions 文件夹
# 选择"上传并部署：云端安装依赖"
```

### 2. 创建数据库集合
```
leaderboard - 排行榜数据
pk - PK房间数据
```

### 3. 配置权限
```
leaderboard: 所有用户可读，云函数可写
pk: 所有用户可读可写
```
