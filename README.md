# 词途小程序

微信背单词小程序

## 功能特点

- 📚 **海量词库** - 35+常用英语单词，按难度分类
- 🎯 **智能学习** - 认识/不认识标记，个性化复习
- 📊 **学习统计** - 实时记录学习进度和掌握情况
- 🎨 **精美UI** - 渐变配色，现代设计风格
- 💾 **本地存储** - 学习记录保存在手机本地
- 🔔 **学习提醒** - 每日定时提醒学习
- ☁️ **API同步** - 支持从后端同步单词数据

## 页面结构

- 首页 - 统计概览和今日推荐
- 学习页 - 单词学习（认识/不认识）
- 个人中心 - 学习记录和统计
- 设置页 - API配置、数据同步
- 提醒页 - 每日学习提醒设置

## 使用方法

1. 下载微信开发者工具
2. 导入本目录
3. 使用测试AppID或申请真机调试

## 目录结构

```
wechat-miniprogram/
├── app.js              # 小程序入口
├── app.json            # 小程序配置
├── pages/
│   ├── index/          # 首页
│   ├── learn/          # 学习页
│   ├── profile/        # 个人中心
│   ├── settings/      # API设置
│   └── remind/        # 学习提醒
├── utils/
│   ├── words.js       # 单词数据和工具函数
│   └── api.js         # API请求模块
└── styles/            # 公共样式
```
