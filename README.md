# 词途小程序

微信背单词小程序

## 功能特点

- 📚 **海量词库** - 35+常用英语单词，按难度分类
- 🎯 **智能学习** - 认识/不认识标记，个性化复习
- 📊 **学习统计** - 实时记录学习进度和掌握情况
- 🎨 **精美UI** - 渐变配色，现代设计风格
- 💾 **本地存储** - 学习记录保存在手机本地
- 🔔 **学习提醒** - 每日定时提醒学习
- ☁️ **API同步** - 支持从后端同步单词数据
- ⭐ **生词本** - 收藏记不住的单词
- 📝 **能力测试** - 选择题形式测试掌握程度
- 🏆 **成就系统** - 解锁学习里程碑
- 🔍 **单词搜索** - 快速查找单词
- 📅 **每日任务** - 每日学习目标
- 📈 **学习计划** - 制定背单词目标

## 页面结构 (14个页面)

- 首页 - 统计概览和今日推荐
- 学习页 - 单词学习
- 个人中心 - 学习记录和统计
- 设置页 - API配置
- 提醒页 - 每日学习提醒
- 生词本 - 收藏单词
- 测试页 - 能力测试
- 分类页 - 按类别学习
- 成就页 - 成就徽章
- 搜索页 - 单词搜索
- 每日任务 - 每日目标
- 学习计划 - 目标规划
- 错题复习 - 复习答错的单词
- 听力练习 - 听音识词
- 成就页 - 成就徽章
- 搜索页 - 单词搜索
- 每日任务 - 每日目标
- 学习计划 - 目标规划

## 使用方法

1. 下载微信开发者工具
2. 导入本目录
3. 使用测试AppID或申请真机调试
