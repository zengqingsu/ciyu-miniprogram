# 词途小程序 v1.6 社交功能技术方案

## 一、功能概述

### 核心功能
1. **排行榜** - 学习数据排名展示
2. **好友PK** - 与好友学习数据对比
3. **群分享** - 分享到微信群/好友

### 数据指标
| 指标 | 计算方式 |
|------|----------|
| 总学习数 | learningRecords.total |
| 连续天数 | learningRecords.streak |
| 掌握单词 | learningRecords.known.length |
| 今日学习 | todayLearnCount |

---

## 二、技术选型

### 方案A：LeanCloud
- 优势：支持多端、实时数据推送
- 劣势：需要额外服务

### 方案B：微信云开发（推荐）
- 优势：免费、微信生态、实时推送
- 劣势：仅微信生态

### 方案C：纯前端（轻量）
- 优势：无后端成本
- 劣势：只能分享固定数据
