# 词途小程序 v1.7 测试验收文档

## 一、部署准备

### 1.1 云开发环境开通
```
微信开发者工具 → 点击"云开发" → 开通环境（免费版即可）
```

### 1.2 云函数上传
```
cloudfunctions 文件夹 → 右键 → 上传并部署：云端安装依赖
```
需上传的云函数：
- getOpenId
- syncUserData  
- getUserData
- updateLeaderboard
- getLeaderboard
- createPK
- joinPK

### 1.3 数据库集合创建
```
云开发控制台 → 数据库 → 添加集合：
— userData（用户学习数据）
- leaderboard（排行榜数据）
- pk（PK房间数据）
```

### 1.4 权限配置
```
leaderboard: 所有用户可读，云函数可写
pk: 所有用户可读可写
```
